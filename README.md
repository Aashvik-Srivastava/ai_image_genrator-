# AI Image Generator 🎨

A simple Python project that generates images from text prompts using Hugging Face Stable Diffusion API.

## 🚀 Features
- Generate image from text prompt
- Saves image locally as PNG
- Displays generated image automatically

## 🛠️ Tech Stack
- Python
- Requests
- Pillow (PIL)
- Hugging Face API

## ⚙️ How it Works
User enters a text prompt → API sends request → AI generates image → image is saved and displayed.

## 🔐 Note
Requires Hugging Face API token stored in environment variables for security.
